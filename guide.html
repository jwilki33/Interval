    Key Recommendations

    1. Start a session before deep work so the curve and distribution reflect real time.
    2. Log friction when you notice it—reflective mode only works if you name the pull.
    3. Glance at the calendar weekly for patterns, not to judge single days.
    4. End sessions cleanly so each run is saved and shows up on the calendar.

    Session Preset

    A slot for how you want to work—length, intent, or structure—before we wire real controls.

    Reflective (current)
    Open-ended; you control start/stop and log distractions by hand.
    Future ideas
    Timed pomodoro-style blocks, “deep work only” reminders, or active tracking hints.

    Focus State

    Same four bands as the stability curve and calendar tinting—read top to bottom as focus slips.

    • Deep — steady, on-task
    • Focused — engaged with mild drift
    • Neutral — attention is split
    • Distracted — pulled away

    Distraction Log

    In reflective mode, the log is built from friction flags: phone, browser drift, or environmental noise.

    Each entry nudges the stability curve and counts toward session distractions—so the log matches what you actually felt pulled you away.

    Active tracking (future) could fill this automatically from screen context.
